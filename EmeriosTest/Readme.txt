El projecto es una aplicacion de consola .net framework 4.8 que lee el archivo "file.txt" alojado en la carpeta "inputs".
Dicho archivo arroja el resultado "D,D,D,D,D"

Otro arhivo "fileBis.txt" arroja "D,D,D,D,D,D,D"

Para cambiar la ruta del archivo a leer, cambiar en app.config key="pathFile".

Correr el .exe para probar la aplicacion.

Saludos.
